# UC03 - Aplicação Web

## O que é esta UC?
Esta Unidade de Competência (UC) envolve o desenvolvimento de uma aplicação web completa para publicação e gerenciamento de conteúdo, integrando frontend, backend e banco de dados.

## Tecnologias
- Frontend: React, HTML, CSS, JavaScript
- Backend: Node.js 
- Banco de Dados: PostgreSQL / MongoDB

## Configuração
1. Criar arquivo `.env` com variáveis de ambiente
2. Configurar banco de dados

## Contribuição
Faça fork, crie uma branch e abra um Pull Request.
